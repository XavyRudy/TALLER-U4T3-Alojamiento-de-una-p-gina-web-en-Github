¿Qué es un Sistema Operativo?

Un sistema operativo (SO) es un software fundamental que permite la interacción entre el usuario y el hardware del computador. Gestiona los recursos del sistema y proporciona una interfaz para ejecutar aplicaciones.

Funciones principales

  • Gestión de procesos
  • Gestión de memoria
  • Control de dispositivos
  • Interfaz de usuario
  • Seguridad del sistema

Tipos de Sistemas Operativos

Los sistemas operativos se pueden clasificar en varios tipos:

  1. Monotarea vs Multitarea
  2. Monousuario vs Multiusuario
  3. Distribuidos
  4. Tiempo real
  5. Moviles

Ejemplos conocidos

Entre los sistemas operativos más utilizados se encuentran:

  • Windows
  • Linux (Ubuntu, Fedora, Debian...)
  • macOS
  • Android
  • iOS
